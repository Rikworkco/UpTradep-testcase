# UpTrader

### Установка проекта
```shell
git clone https://github.com/AikoSora/UpTrader.git
```

Установите необходимый пакет для работы:
```shell
pip install django
```

Создайте и присоеденитесь к БД:
```shell
python manage.py migrate
```

### Запуск проекта

Для запуска введите команду:
```shell
python manage.py runserver
```
